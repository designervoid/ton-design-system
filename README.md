# TON Design System

## Prerequisites

`node16.13.2`, `npm8.1.2`, `make3.81`

## Quick start

Install packages:

```zsh
npm i
```

Run `dev` (1) or `prod` (2) environment

1:

```zsh
npm run dev
```

2:

```zsh
make prod
npx http-server dist
```

`

  /*
